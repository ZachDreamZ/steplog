# Changelog

## 0.1.0 (2026-07-20)

- Initial release
- `step()` context manager with timing and success/failure indicators
- `output.info()`, `.success()`, `.warn()`, `.error()` message helpers
- `output.header()`, `.subheader()`, `.section()` structural helpers
- `output.table()` formatted table output
- `output.code()`, `.json()`, `.raw()`, `.divider()` utilities
- `Spinner` / `spinner()` for long-running operations
- Zero external dependencies
